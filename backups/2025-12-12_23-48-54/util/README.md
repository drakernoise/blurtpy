# Util Directory

This directory contains utility scripts primarily used for Continuous Integration (CI) and Continuous Deployment (CD).

## Contents

*   **`package-*.sh`:** Shell scripts for packaging the library for different operating systems (Linux, OSX).
*   **`travis_*.sh`:** Scripts related to Travis CI configuration (legacy).
*   **`appveyor/`:** Configuration for AppVeyor CI (Windows).

## Usage

These scripts are intended for maintainers and automated build systems. They are not required for normal usage of the library.

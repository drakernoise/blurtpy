# Documentation Index

This directory contains additional documentation, guides, and reports for the `blurtpy` project.

## Files

*   **[nodes.md](nodes.md):** A list of public RPC nodes for the Blurt blockchain, including instructions for automatic and manual selection.
*   **[walkthrough.md](walkthrough.md):** A comprehensive guide covering migration from legacy libraries, installation, and common usage scenarios.
*   **[security_audit_report.md](security_audit_report.md):** Details of the security audit performed on the library, highlighting fixes and improvements.
*   **[security_audit_plan.md](security_audit_plan.md):** The original plan and checklist for the security audit.

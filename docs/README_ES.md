# Índice de Documentación

Este directorio contiene documentación adicional, guías y reportes para el proyecto `blurtpy`.

## Archivos

*   **[nodes.md](nodes.md):** Una lista de nodos RPC públicos para la blockchain de Blurt, incluyendo instrucciones para selección automática y manual.
*   **[walkthrough.md](walkthrough.md):** Una guía completa que cubre la migración desde librerías legacy, instalación y escenarios de uso comunes.
*   **[security_audit_report.md](security_audit_report.md):** Detalles de la auditoría de seguridad realizada a la librería, destacando arreglos y mejoras.
*   **[security_audit_plan.md](security_audit_plan.md):** El plan original y lista de verificación para la auditoría de seguridad.

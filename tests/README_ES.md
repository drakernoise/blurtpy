# Directorio de Tests

Este directorio contiene la suite de pruebas para `blurtpy`. Incluye pruebas unitarias y de integración para asegurar la estabilidad y corrección de la librería.

## Estructura

*   **`blurtpy/`:** Pruebas específicas para el core y módulos de `blurtpy`.
*   **Carpetas Legacy (`beem*`):** Pruebas heredadas del fork original, mantenidas para pruebas de regresión de componentes base.

## Ejecutar Pruebas

Puedes ejecutar las pruebas usando `pytest` desde el directorio raíz:

```bash
pytest
```
